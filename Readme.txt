iHack idea 

Project - Medizin Website :
All Three Modules made using different web technologies:
1. PHP for Medizin Login and Registration
2. Django for Medizin Website and AI Expert
3. Flask for Medizin Chatbot

To Run PHP Module 1 :
# Install Xampp, Open and Start Apache, MySQL server
# Go to Xampp/xampp/htdocs directory in C:/ drive
# Copy paste the Medizin Login and Registration source folder there
# Go to Google Chrome and type "localhost/Medizin Login and Registration/ulogin.php"
# Explore the Module features

To Run Django Module 2 :
# Install Anaconda Navigator,Django and Set up Django Virtual Environment
# Open Anaconda prompt and go to Medizin Website and AI Expert directory
# Type python manage.py createsuperuser and create admin credentials
# Type python manage.py runserver
# If any Module not found in Laptop, Install using pip install module_name
# Else go to "http://localhost/127.0.0.1/8000" and Explore Module features

To Run Flask Module 3 :
# Install Anaconda Navigator,Flask, MongoDB and all packages mentioned in Medizin Chatbot/requirements.txt
# Set up Virtual Environment for Flask
# Create MongoDB database,Pusher account and get their credentials asked in app.py
# Fill these credentials in agent.js and script.js
# Create Collection named "Medizin" in MongoDB
# Insert Medizin Website and AI Expert/mongodb_doc.txt content as document in thee collection
# Open Anaconda Prompt, Go to Medizin Chatbot directory and type python app.py
# If any Module not found in Laptop, Install using pip install module_name
# Else go to "http://localhost/127.0.0.1/5000" to explore Patient Side features
  and go to "http://localhost/127.0.0.1/5000/agent" to explore Doctor Side features
  of Medizin Chatbot
